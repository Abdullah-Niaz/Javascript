Event Propagation

  • Event Propagation refers to the way, events flow through the DOM. When an event is triggered on a particular element

      • There are two ways of event Propagation:

    1. Event Structing
      • The event start from the root of the document and travel to down to the target eleemnt
    2. Event Bubbling
      • The event start from the target element, it bubbles back up to the root
